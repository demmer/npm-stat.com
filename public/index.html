juttle npm-stat

This site is a fork of npm-stat.com powered by Juttle.
See how we built it.

npm-stat is a small utility that generates charts of the statistics for any package in npm. Just enter the package or author name in the form below.

These statistics are not provided in real-time. All numbers will change once a day.

If there are other issues with the service, please send me an email to or create an issue at Github.

The charts of this service are powered by Highcharts JS and of course npm.
